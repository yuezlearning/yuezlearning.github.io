A brand new certificate built in Open edX with custom reusable components and third-vendor activities that ship across the catalog and save 7 Taps subscriptions.

01Overview

WGU Academy is planning to launch a Generative AI Skills Certificate to equip learners with practical AI literacy — from prompt engineering fundamentals to responsible AI use in professional contexts.

As the Instructional Technologist, I was responsible for building the entire course infrastructure in Open edX: authoring interactive content, creating reusable custom components, integrating third-party tools via LTI, and ensuring a consistent, engaging learner experience across all modules.

03Collaboration

Building an online certificate requires tight cross-functional coordination. I worked daily with four key roles:

Learning Experience Designer

Translated pedagogical goals into course structure, learning objectives, and assessment alignment. I worked with them to ensure the technical build matched the instructional intent.

Multimedia Designer

Created video content, graphics, and visual assets. I integrated these into Open edX with proper responsive containers and accessible alternatives.

Assessment Developer

Designed formative and summative assessments. I built the technical layer — rubric scorers, quiz integrations, and CodeSignal LTI connections for coding assessments.

Program Manager

Coordinated timelines, stakeholder reviews, and launch sequencing. I provided build estimates and flagged technical blockers early.

04Components

A key contribution was building custom, reusable components that replaced expensive third-party tools (7 Taps) while providing a better student experience. These components are now shared across the entire WGU Academy catalog.

Step Cards

Interactive step-by-step cards combining text and animated GIF walkthroughs to guide learners through multi-step processes in a clear, visual format.

Chatbot Interface

A simulated chatbot interaction combining text and images that lets students practice conversational AI prompting within the course flow.

Carousel

This component replaces the 7 Taps component, combining text and animated GIF content in a swipeable carousel format for a more engaging inline experience.

Why Custom Over 7 Taps?

  • • Limited customization

After (Custom Components)

  • • Fully branded & customizable

05CodeSignal

CodeSignal is used for supplementary content and activities to enhance the mastery of knowledge, providing students with hands-on coding practice directly within the course.

Integration Details

01

LTI 1.3 Configuration

Set up secure OAuth2 handshake between Open edX and CodeSignal, enabling seamless single-sign-on and grade passback.

02

Assessment Mapping

Mapped each course module to specific CodeSignal challenges, ensuring progressive difficulty and alignment with learning objectives.

Inline CodeSignal activities embedded within the course to reinforce learning through practice.

06Experience

Every technical decision was made with the student experience in mind. Key improvements over the previous course-build approach:

Inline interactivity

No more redirecting to external tools. Students stay in the learning flow — prompt sandboxes, scenario branchers, and coding challenges all load within the page.

Consistent design language

Reusable components share the same interaction patterns, visual style, and feedback mechanisms. Students learn the interface once and transfer that knowledge across modules.

Immediate feedback

Components provide in-context responses — hints on prompt quality, consequences of scenario choices, rubric scoring breakdowns — without waiting for instructor grading.

Accessible by default

All custom components built with keyboard navigation, screen reader support, and WCAG AA color contrast. No student is excluded from the interactive experience.

07Reflection

Build vs. buy decisions

The 7 Taps replacement proved that investing dev time upfront saves exponentially on subscription costs at scale. But the real win was quality control — we own the experience end to end.

Cross-functional fluency matters

Speaking the language of instructional designers, multimedia creators, and assessment specialists made me a better engineer. I learned to ask "what's the learning goal?" before "what's the tech solution?"

Components as a product

Treating reusable components like an internal product — with documentation, versioning, and stakeholder demos — dramatically increased adoption across the catalog. Other course teams now request them by name.
